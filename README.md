# ISM2411 Data Cleaning with GitHub Copilot

This project demonstrates a simple Python data cleaning pipeline using pandas and GitHub Copilot assistance.

## Project Goals

- Load messy sales data
- Clean inconsistent formatting
- Handle missing values
- Remove invalid records
- Export a cleaned CSV file
